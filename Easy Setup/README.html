Easy Setup

Installer to install anything in one click

Install instructions

  1. Run the file called Easy Setup.exe
  2. Click Install
  3. Wait until it installs the software

Making Easy Setup for a program

Distributing Easy Install for your program is not difficult, you need only to edit some files and strings in the code, and compile the installer.

Requirements:

  • Visual Studio 2019 or Visual Studio 2022
  • .NET Framework component
  • Installer built in MSI package (to build a msi installer we reccommend InstallShield)

Instructions:

  1. Download the source code from GitHub
  2. Open the solution called Easy Setup.sln in Visual Studio
  3. Edit the following strings:

        soft_name = ""; - Put here the software name

        author  = ""; - Put here the author name

        version = ""; - Put here the program version

        require_admin = true; - Set the value to true if the installer requires admin privileges (default) or to false if not

     4. Paste your msi setup package in the "files" directory. Delete the setup.msi provided by default and rename the new msi package to "setup.msi"

     5. Compile the setup and enjoy
